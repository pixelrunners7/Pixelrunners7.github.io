About the Game

Pixel Runners: Time Shift is a fast-paced pixel runner where you don’t just run – you control time!

Use unique time manipulation mechanics to overcome traps, dodge obstacles, and discover new paths in a procedurally generated city. Get ready for fast-paced action, reflex-testing gameplay, and a stunning retro pixel-art aesthetic.
